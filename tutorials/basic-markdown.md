# Основы Markdown

Изучите базовый синтаксис Markdown для создания красивого контента.

## Заголовки

```markdown
# H1 Заголовок
## H2 Заголовок  
### H3 Заголовок
```

## Форматирование текста

- **Жирный текст**: `**жирный**`
- *Курсив*: `*курсив*`
- `Код`: `` `код` ``
- ~~Зачеркнутый~~: `~~зачеркнутый~~`

## Списки

### Маркированный список
- Элемент 1
- Элемент 2
  - Подэлемент 2.1
  - Подэлемент 2.2

### Нумерованный список
1. Первый пункт
2. Второй пункт
3. Третий пункт

## Ссылки и изображения

- Ссылка: `[текст](URL)`
- Изображение: `![alt](URL)`
